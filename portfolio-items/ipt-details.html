Interactive Periodic Table

Course Project - Mobile Application

  • Developed: Fall 2022
  • Project URL: GitHub Repository
  • Language(s): Kotlin
  • Technologies used: Android Studio
  • Architecture: Model-View-ViewModel
  • Collaborators: Robert Mason and Wesley Parker

The Interactive Periodic Table app is a mobile application that served as a final course project for "Mobile Programming". My team decided to develop a mobile app that will serve as an interactive periodic table. The purpose of this application is to engage chemistry students in active learning when learning about the elements of the periodic table. A useful resource for this project was the UAteach program, as every member on the team was a part of the program at one point. We utilized the program to assist us in creating ideas for the app.

The application opens to a simple periodic table layout. The user may tap on one of the element cells to open a infobox pertaining to the element. The app also includes a Compounds page, as well as a Gems page. Here, a user is given a list of buttons labeled with a type of gem or compound. When tapped, certain element boxes are blocked out and a formula is shown. This tells the user what that gem/compund is composed of. This application does well with providing quick information to students, and by letting them associate chemical elements with everyday items.
