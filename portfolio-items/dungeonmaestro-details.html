DungeonMaestro5e

Web Application

  • Started: March 2024
  • Language(s): TypeScript, HTML, CSS
  • Technologies used: Angular, Firestore
  • Collaborators: Zack Bailey

DungeonMaestro5e is a web application project currently in development. The purpose of this app is to serve as a wiki for homebrewed campaigns. Here, users can log on and access saved campaigns as either a DM or Player. A DM can manage information in a campaign and add notes for particular players. Players will have only knowledge of the world that their character would have. The purpose of DungeonMaestro5e is to provide a simple platform to keep track of game progress, mitigate metagaming, and allow players to utilize information to enhance their roleplaying.
